# Blockchain | 区块链

与其说比特币是区块链技术的第一个应用，不如说区块链技术是为了实现数字货币比特币，而出现的一种重组现有技术得到的“新”技术。
至于除了数字货币之外还能实现什么应用，我想，等比特币落地的那天，一切都会揭晓。

## Bitcoin | 比特币（区块链 v1.0）

比特币是一种试验性的数字货币，能够让世界上任何地方的任何人进行即时支付。
比特币使用点对点的技术，在没有中央权力的情况下运作：管理交易和发行货币通过网络共同完成。
